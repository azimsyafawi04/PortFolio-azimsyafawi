# SSCSS-Smart-Campus

Secure Student Information & Identity Management System (SSIIMS) prototype repository.

## Project Description
This repository contains a functional web prototype (MVP) demonstrating security-by-design principles and Role-Based Access Control (RBAC) in a Smart Campus context.

## Features
- **Secure Login View**: Fully simulated multi-role authentication with demo credential assistance.
- **Student Portal**: Secure profile editing, dynamic course enrollment, and RBAC-verified grade reports.
- **Admin Console**: Dynamic account provisioning, MFA configuration toggling, account revocation, and real-time security audit logging.
- **Security-by-Design Annotation**: Inline comments detailing production-ready backend checks (rate limiting, CSRF tokens, input sanitization, JWT authorization middleware, syslog integration).

## Tech Stack
- HTML5 (Semantic elements)
- CSS3 (Vanilla glassmorphism style sheet)
- Vanilla JavaScript (Single Page Application architecture)
